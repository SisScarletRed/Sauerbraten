UI Improvements

Commands

Some commands changed, and some came new. Here is a list of these changed/new ones:

  • guibutton: now taking four strings, the 4th one declares hoverinfo
  • guitext: now taking three strings, the 3rd one declares hoverinfo
  • guiplayers: list the players
  • guiserverextinfo: show serverpreviews, if available
  • guitextscale [float] [code]: will set the text scale to [float], execute [code] and set textscale back after.
  • guibackgroumd [color]: sets the background for the guilist
  • guititle: another hoverinfo declaration
  • guititleleftalign: same as guititle
  • guiradio: another hoverinfo declaration
  • guibitfield: another hoverinfo declaration
  • guicheckbox: another hoverinfo declaration
  • guifont [font] [body]sets the font to [font], executes [body] and goes back to old font again

Mainmenu - Example

To the left you can see my mainmenu. For one, the mouse turns to a hand, when hovering something clickable. Also, a hoverinfo appears, if defined, and the text color of the hovered item is cyan. And a new guiskin-image is there, looking way cleaner than before.
