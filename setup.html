Installation

A step by step guide how to install my client

Install only once

Only download the client once. You won't be up-to-date with the newest version, but if you want to update, you need to reinstall everything like descriped here.
Have https://github.com/theonlyrealquality/Sauerbraten opened.

  • Download the .zip file of the Repository
  • Extract the compressed folder to e.g. "QuEd".
  • Download the release for your architecture (32bit or 64bit, if unsure download both) and copy them into their folders (bin32 or bin64)
  • Now either download the SVN version of Sauerbraten and copy it's packages folder to your "QuEd" folder, or copy the normal Sauerbraten packages folder. If you use the SVN version, make sure the maps are still the old ones, and if you use the release version, be prepared for not being able to use brushes and getting ~20 file not found messages on every startup.
  • Hit sauerbraten.bat to start the game
  • Your game files that were in My Games/Sauerbraten before now are in "Qued"/home and "QuEd"/home/saved.

Keep it up-to-date

Note: When there are new features in menus, but the executables were not updated yet, there might be some consolespam about unkwown command or similar. It should still be functioning though.

  • Have a GitHub account and a GitHub Client installed.
  • Log in to GitHub in your favorite browser and hit "Clone in Desktop".
  • Clone it
  • Download the executables as descriped above.
  • Copy the packages folder as descriped above.
  • Play.
  • If you want to contribute, fork this project and create a pull request on GitHub.
